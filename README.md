# Plataforma de instituciones educativas

Base de desarrollo para una solución integral orientada a colegios, escuelas, academias e institutos.

## Estado actual
Proyecto en fase de bootstrap arquitectónico y técnico.

## Estructura
- `apps/web` → frontend
- `apps/api` → backend
- `packages/*` → módulos compartidos
- `infrastructure/*` → despliegue y operación
- `docs/` → documentación funcional y técnica
- `scripts/` → automatizaciones del proyecto
